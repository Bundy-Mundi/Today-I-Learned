# Array as Abstract Data Type

## ADT

### - Representation of Data

1. Array Space

2. Size

- Literally a size of an array

3. Length

- Number of elements in an array

- Example

    ```cpp
    int X[5];
    // X size: 5
    // X length: 0
    ```


### - The Set of Operations of Data

1. Display()

2. Add(x) / Append(x)

3. Insert(index x)

4. Delete(index)

5. Search(x)

6. Get(index)

7. Set(index, x)

8. Max() / Min()

9. Reverse()

10. Shift() / Rotate()

### Two ways of creating an Array

1. Stack

2. Heap
